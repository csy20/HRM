# HRM

## Mobile UI
- Menu: On phones, tap "Menu" to open the sidebar.
- Navigation: Selecting a unit auto-closes the sidebar.
- Safe area: Layout supports notched screens (`viewport-fit=cover`).
- Spacing: Cards and charts use tighter spacing on small screens.

## Local Preview
Use your host browser to open the file directly:

```bash
"$BROWSER" /workspaces/HRM/index.html
```

If `$BROWSER` isn't set, open the file path in your browser manually.
